 
1️⃣ What is the difference between var, let, and const?

var is like a normal variable you can change anywhere in a function, let is block-scoped and changeable, const is block-scoped and can’t change.

2️⃣ What is the spread operator (...)?

The spread operator ... expands an array or object into individual elements,


3️⃣ What is the difference between map(), filter(), and forEach()?

forEach loops without returning, map returns a new array of modified items, filter returns a new array of items that pass a test.


4️⃣ What is an arrow function?

An arrow function is a shorter syntax for writing functions using ()=> and it doesn’t have its own this.


5️⃣ What are template literals?

Template literals are strings using backticks ` that allow variables and expressions inside ${}.
